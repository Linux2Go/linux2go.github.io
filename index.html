Some of the amazing products that we use

They might be just what you need.

Consul

Consul provides service discovery, monitoring, and a distributed key/value store for information exchange.

Jenkins CI

Jenkins is the gold standard for Continuous Integration systems. Hundreds of plugins and customizable to your hearts content, it's central to thousands of organizations' QA process.

aaSemble

Toolkit for deploying and updating complex applications for development as well as for production.

Ubuntu

The leading operating system in the cloud and an extremely popular choice for containers as well.

Alpine Linux

Security-oriented, lightweight Linux distribution. An increasingly popular choice for containers.

Docker

The de-facto standard for running containers. Things like Kubernetes are built on top of Docker.

AWS

Leading provider of cloud services, from basic infrastructure like virtual machines and storage to higher level services like messaging, autoscaling, etc.

Digital Ocean

Infrastructure-as-a-Service (IaaS) provider, very popular with developers. Fast, straightforward and competitively priced.
